## 0.0.1

* Initial release.
* Supported field types: text, number, date, dropdown, textarea, image, custom.
* Cascading dropdowns with auto-reload and auto-clear.
* Static and async dropdown sources via `DropdownSource` union type.
* Built-in required validation + custom validator functions.
* Conditional field visibility via `FieldDependency`.
* External `FormController` (pure `ChangeNotifier`, zero dependencies).
* Pre-fill support for edit and clone modes.
* View-only mode for read-only rendering.
* `RjFormTheme` for full visual customization.
* Zero external dependencies (only `flutter` SDK + `image_picker`).
